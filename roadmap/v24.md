# Roberto v24 Roadmap

## Objective
Improve everyday usability with preset AI actions and impact-first shortcuts.

## Scope

### 1) AI Actions
- `actions run --name one-issue`
- `actions run --name challenge-thesis`
- `actions run --name build-counter`
- `actions run --name impact-top`

### 2) One-Issue Clarity
- `one-issue` action pulls highest-signal briefing delta with citations.
- Reuses muted/snoozed and ranking controls from prior versions.

### 3) Retrieval Hardening
- Index Greene cards as first-class search docs (`kind=card`).

## Acceptance Criteria
- Action outputs are short, actionable, and cited.
- `one-issue` is deterministic from local cache.
- Card search supports quick retrieval for writing flow.
