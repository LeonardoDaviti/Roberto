# Roberto v23 Roadmap

## Objective
Turn cards into traceable drafts via output studio modes.

## Scope

### 1) Draft Modes
- `draft generate --mode memo`
- `draft generate --mode brief`
- `draft generate --mode essay-skeleton`
- `draft generate --mode chapter-draft`
- `draft generate --mode compile`

### 2) Citation Traceability
- Non-compile modes keep per-section source footers.
- Compile mode strips scaffolding for clean read.

### 3) Persistence
- Draft notes saved under `notes/greene/drafts/`.
- SQLite table: `studio_outputs` for mode/topic/output linkage.

## Acceptance Criteria
- Draft generation works from keeper cards/chapter candidates.
- Output remains source-backed before compile mode.
- Studio outputs are reproducible from local state.
