# Roberto v6 Roadmap

## Objective
Shift from passive summarization to active thinking support with recombinable, source-backed memory artifacts.

## Scope

### 1) Idea Cards (Greene-style notecards)
- Generate per-user idea cards from validated notecards.
- Each card proposes one `essay|product|experiment`.
- Every card stores explicit tweet citations.
- Persist cards in SQLite (`idea_cards`) and publish per-user idea notes in `notes/ideas/`.

### 2) Conflict Cards (contradiction-first)
- Detect disagreements across users on overlapping tags.
- Keep both claims separated (no blending).
- Persist to `conflict_cards`.
- Publish rolling conflict note at `notes/conflicts/latest.md`.

### 3) Weekly Shuffle Pack
- Build a weekly pack of 12 diverse cards.
- Add 3 non-obvious cross-tag connections.
- Enforce citation gate: each connection includes source tweet refs.
- Publish at `notes/shuffles/<YYYY-Www>.md`.

## Definition Of Done
- v1/v2 runs generate/update idea cards and idea notes.
- Conflict note exists and remains source-backed.
- Weekly shuffle note is created/updated with cited connections.
- All new note types use marker-only overwrite behavior.

## Out Of Scope (v7)
- Graph-level entity resolution and story/entity timeline queries.
