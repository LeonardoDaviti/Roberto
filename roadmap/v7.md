# Roberto v7 Roadmap

## Objective
Make cross-user synthesis durable with an explicit entity graph and timeline queries.

## Scope

### 1) Entity Index + Alias Resolution
- Add canonical `entities` table and `entity_aliases`.
- Extract entities from tweets and story text/tags.
- Resolve aliases to canonical IDs for stable querying.

### 2) Graph Links
- Link entities to tweets/stories via `entity_links`.
- Link stories to entities via `story_entities`.
- Keep links incremental and idempotent per run.

### 3) Timeline Views
- Add CLI:
  - `entity list`
  - `entity show <query> --days <N>`
- Add per-entity timeline notes in `notes/entities/<entity_id>.md`.
- Extend story ops with `stories show <slug>` to inspect linked sources/entities.

## Definition Of Done
- New tweets and digest stories update the entity graph each run.
- `entity show NVIDIA --days 90` returns source-backed timeline rows.
- Entity notes are created/updated through marker-only writes.
- Story ↔ entity links are queryable from CLI.

## Out Of Scope (v8)
- Full NER model integration.
- Knowledge graph ranking / graph embeddings.
- UI layer on top of entity-story timelines.
